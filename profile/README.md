# Kolmar-KAIST Research

콜마-KAIST 공동 연구 레포지토리입니다.  
혼합물(Mixture) 기반 물성 예측 모델과 시스템 개발을 목표로 합니다.

---

## Core Repositories

연구의 기반이 되는 구성 요소입니다.

- **[Kolmar Learning](https://github.com/kolmar-korea/KolmarLearning)**: 학습 전략 및 트레이닝 파이프라인  
- **[Kolmar Models](https://github.com/kolmar-korea/KolmarModels)**: 모델 아키텍처 정의  
- **[Kolmar Data](https://github.com/kolmar-korea/KolmarData)**: 데이터셋 구조 및 전처리  

---

## Projects

Core 레포를 기반으로 한 연구 및 응용 프로젝트입니다.

- **[KolmarApp-2025](https://github.com/kolmar-korea/KolmarApp-2025)**: 2025 KAIST 프로젝트 애플리케이션  
- **[Mixing-Former](https://github.com/kolmar-korea/MixingFormer)**: 혼합물 표현 학습 모델 연구  
- **[Kolmar-Blend](https://github.com/kolmar-korea/KolmarBlend)**: 물성 레이블 통합 모델 개선
